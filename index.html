Dart for Windows

Community Installer

The installer includes the Dart SDK which is necessary for development and Dartium, a browser that runs native Dart code.

To update Dart, just run the Dart Update program.

Fully tested, ready for production
Early access to new features
This is a community-supported installer. It is not supported by the Dart project authors. The source is available on GitHub. Download 32-bit versions here Stable 32-bit and Dev 32-bit.
